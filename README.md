# Intranet LABJOR

Sistema web desenvolvido em PHP e MySQL para gerenciamento interno.

## Funcionalidades

- Controle de usuários
- Reserva de equipamentos
- Agenda de salas
- Integração com Keycloak (Autenticação Central Unicamp)

## Tecnologias

- PHP
- MySQL/MariaDB
- JavaScript - com biblioteca JQuery
- Bootstrap

## Instalação

1. Clone o repositório
2. Importe o banco de dados
3. Copie os arquivos *_example
4. Configure as credenciais
5. Execute em Apache/Nginx

## Estrutura

- agenda-salas/
- reserva-equipamentos/
- gerenciar-pessoas/

## Observações

Dados sensíveis e credenciais foram removidos do repositório.

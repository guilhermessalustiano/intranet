# Intranet

Sistema web de Intranet desenvolvido durante o estágio no Labjor entre abril de 2024 e abril de 2026. Possui módulos de agendamento, reserva de equipamentos e gerenciamento de cadastros. É integrado à Autenticação Central da Unicamp, mas pode ser implementado login comum.

## Funcionalidades

- Controle de usuários
- Reserva de equipamentos
- Agenda de salas
- Integração com Keycloak (Autenticação Central Unicamp)

## Tecnologias

- PHP
- MySQL/MariaDB
- JavaScript - com biblioteca JQuery
- Bootstrap

## Instalação

1. Clone o repositório
2. Importe o banco de dados
3. Copie os arquivos *_example
4. Configure as credenciais
5. Execute em Apache/Nginx

## Estrutura

- agenda-salas/
- reserva-equipamentos/
- gerenciar-pessoas/
